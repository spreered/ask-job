# 有事求神，沒事求人
有事求神，沒事求我
這是非官方事求人機器人
「人員類別」沒有「約僱人員、駐外人員、代理教師、代課教師、實習老師、聘用人員」(不含公務人員)
一切資料以人事行政局事求人網站為主
